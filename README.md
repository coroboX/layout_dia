## Used technologies:

- HTML5 / CSS3 / JS
- BEM
- eslint, stylelint, linthtml
- Sass preprocessor
- NPM, Gulp
- Git


## How to run project locally:
1. `git clone` or download project via Git.
2. Run `npm install` (or just `npm i`)
3. Run `npm start`.
